# 🚀 TaskFlow – React Task Manager

TaskFlow is a clean and responsive task management web application built using React and Tailwind CSS. It helps users manage daily tasks efficiently with features like filtering, searching, sorting, and dark mode.

---

## ✨ Features

- ✅ Add, edit, and delete tasks
- ✅ Mark tasks as completed or pending
- 🔍 Real-time search functionality
- 🎯 Filter tasks (All / Completed / Pending)
- 📅 Sort tasks by due date
- ⚡ Priority levels (Low / Medium / High)
- 💾 Persistent storage using LocalStorage
- 🌙 Dark / Light mode toggle
- 📭 Empty state UI

---

## 🛠 Tech Stack

- React (Vite)
- Tailwind CSS
- JavaScript (ES6+)

---

## 🧠 Concepts Used

- React Hooks (useState, useEffect)
- Controlled Components
- Conditional Rendering
- State Management
- Array Methods (map, filter, sort)
- LocalStorage Integration

---

## 📦 Installation

```bash
git clone https://github.com/shauryasync/taskflow-react.git
cd taskflow-react
npm install
npm run dev
```
